# MSML606 HW4: Randomized Quicksort & Heapsort

## Objective
Implement **Randomized Quicksort** and **Heapsort** (no built-in sort/heap packages). Analyze performance on various inputs.

## Given Files
- `Homework4.py` - Sorting implementations boiler plate 
- `testcases.csv` - Sample inputs  
- `README.md` - This file
- `MSML606 HW4` - Assignment instructions

## Expected Submission Files
- `Homework4.py` - Source code with your sorting implementations and test cases
- `report.pdf`or  `report.docx` - Analysis and results for p1 and p2

## Tasks Part 1:
- Implement both algorithms from scratch  
- Test on random, repeated, ordered, and reverse inputs  
- Measure runtime (`time.perf_counter()`) and memory (`tracemalloc`)  
- Handle exceptions (e.g., `RecursionError`)  
- Document findings in the report

## Part 1 Rubric (40 pts)
- Quicksort and Heapsort Implementation (10)
- Test Cases (10)
- Performance analysis (5)
- Report: description, complexity, comparison, trends (15)

## Tasks Part 2:
- Randomized Quicksort Tracing
- Randomized Quicksort Recurrence Relations
- Randomized Quicksort Recursion Tree and Solving the Recurrence
- Heapsort Tracing

## Part 2 Rubric (8 pts)
- Randomized Quicksort Tracing (3)
- Randomized Quicksort Recurrence Relations (1)
- Randomized Quicksort Recursion Tree and Solving the Recurrence (2)
- Heapsort Tracing (2)

python Homework4.py
